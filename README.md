# Raylib-Base
Basic setup for Raylib and ImGui, using Cmake for building.

# Setup Guide:
* Open the folder with the CMakelists.txt in Visual Studio
* Open the CMakelists.txt and press Ctrl + S
* Select the startup item (mygamecpp.exe)
* Press Ctrl + F5 to run the project or F5 for debug mode
* If you ever get weird errors, then delete the "out" folder and Crtl + S on the CMakelists.txt again
